# Análisis de ventas  Challenge Alura Store
 
 ## Sobre el proyecto

Este proyecto es parte del Challenge de Data Science de Alura Latam.
El objetivo es analizar las ventas de cuatro tiendas para decidir cuál debería vender el Sr. Juan.

El análisis se realizó usando Python en Google Colab, aplicando conceptos básicos de análisis de datos.

 ## Herramientas usadas

Python

Pandas

Matplotlib

Google Colab

 ## Qué se analizó

Durante el proyecto se analizaron:

Ingresos totales por tienda

Ventas por categoría

Calificación promedio de los clientes

Productos más y menos vendidos

Costo de envío promedio

También se hicieron gráficos simples para comparar los resultados.

 ## Cómo usar el proyecto

Abrir el notebook en Google Colab

Ejecutar las celdas en orden

Revisar los resultados y gráficos

 ## Conclusión

Después del análisis, se recomienda vender la Tienda 4, ya que tiene el menor ingreso total en comparación con las otras tiendas y un costo de envío promedio más bajo, sin grandes diferencias en calificaciones.
